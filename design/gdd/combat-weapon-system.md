# 战斗武器系统 (Combat & Weapon System)

> **Status**: Draft
> **Author**: Game Designer
> **Last Updated**: 2026-05-07
> **Implements Pillar**: 自然与人的协力

## Summary

玩家通过鼠标点击操控守卫者角色攻击怪物，金币购买4种武器（普通剑→火焰大剑→寒冰大剑→冰火法杖），第5波自动赠予灭世之剑。每种武器有独特视觉特效和机制，增加战斗趣味性。

> **Quick reference** — Layer: `Core` · Priority: `MVP` · Key deps: `economy-system.md, wave-system.md, weapon-vfx-system.md`

---

## Overview

战斗系统是玩家最直接的游戏交互。玩家点击怪物发起攻击，守卫者角色飞向目标施加伤害。随着金币积累，玩家解锁更强大的武器，每种武器有独特的打击感和范围效果。武器系统和生态系统互补：武器输出点伤，生态系统提供持续防御。

---

## Player Fantasy

玩家应感受到：随着武器升级，从一个普通守卫成长为掌控冰与火的魔法战士，最终手持灭世之剑力斩污染魔王，每一次挥剑都有十足的爽快感。

## Detailed Design

### Core Rules

**守卫者 (Guardian)**
1. 守卫者角色位于领地中部，初始显示在核心左侧
2. 玩家**左键点击**一个怪物：守卫者飞向该怪物，执行一次攻击，然后返回
3. 守卫者移动速度：`GUARDIAN_SPEED = 200 px/s`
4. 守卫者同时只能攻击一个目标（攻击中不可切换目标）
5. 守卫者不可被怪物攻击（玩家角色无血量，以核心血量代替玩家血量）
6. 攻击冷却：每次攻击完成后有 `BASE_ATTACK_COOLDOWN = 0.5s` 冷却

**武器系统**
1. 初始武器为普通剑，不需要购买
2. 武器只能持有一种，购买新武器自动替换旧武器
3. 武器效果立即生效，无需额外确认
4. 武器购买界面在左下角商店按钮打开，波间自动暂停不存在（玩家需自行管理时机）

### 武器规格表

| 武器 | 获取方式 | 价格 | 伤害 | 攻击范围 | 特殊效果 | 特效描述 |
|------|----------|------|------|----------|----------|----------|
| 普通剑 (Basic Sword) | 初始拥有 | 免费 | 25 | 单体 | 无 | 白色剑光挥砍 |
| 火焰大剑 (Flame Greatsword) | 金币购买 | 150金 | 40 | 单体+近身溅射30px | 燃烧：目标持续3s受5/s伤害 | 橙红色剑光+火焰粒子轨迹，击中爆炸 |
| 寒冰大剑 (Frost Greatsword) | 金币购买 | 200金 | 35 | 单体+减速 | 冻结：目标移动速度-50%持续3s | 蓝白色剑光+冰晶粒子，击中冰霜效果 |
| 冰火法杖 (Ice-Fire Staff) | 金币购买 | 300金 | 30×2 | 远程180px+分裂弹 | 发射冰火双弹，各自带冻结和燃烧效果 | 法杖发射双色能量球，分裂时色彩绚丽 |
| 灭世之剑 (World-Ender Sword) | 第5波开始自动赠予 | 免费 | 120 | 全屏扇形150°范围 | 一击清场小怪；对Boss造成双倍伤害 | 金银双色巨大剑气横扫全屏，天崩地裂特效 |

### States and Transitions

**守卫者行为状态**

| State | Entry Condition | Exit Condition | Behavior |
|-------|----------------|----------------|----------|
| 待机 (Idle) | 初始 / 攻击返回后 | 玩家点击目标 | 守卫者站立待机动画 |
| 移动 (Moving) | 玩家点击目标 | 到达目标位置 | 飞向目标，播放移动动画 |
| 攻击 (Attacking) | 到达目标位置 | 攻击动画完成 | 播放武器攻击动画，触发伤害和特效 |
| 返回 (Returning) | 攻击完成 | 返回领地中心 | 飞回原始位置 |
| 冷却 (Cooldown) | 返回完成 | 冷却时间结束 | 待机状态，点击无效，显示冷却指示 |

### Interactions with Other Systems

| System | Interaction |
|--------|-------------|
| 金币经济系统 | 购买武器时消耗金币 |
| 波次入侵系统 | 攻击波次中的怪物，减少怪物HP |
| 武器特效系统 | 每种武器触发对应的视觉特效 |
| 怪物AI系统 | 燃烧/冻结效果修改怪物的移动速度和持续受伤状态 |

## Formulas

### 实际伤害计算

```
actual_damage = weapon_base_damage * (1 + combo_bonus)
```

| Variable | Type | Range | Source | Description |
|----------|------|-------|--------|-------------|
| weapon_base_damage | int | 25-120 | 武器配置 | 武器基础伤害 |
| combo_bonus | float | 0.0-0.5 | 连击计数 | 连续3次击杀同类怪物+50%伤害 |

**Expected output range**: 25（普通剑无连击）到 180（灭世之剑满连击）

### 燃烧总伤害

```
burn_total = BURN_DPS * BURN_DURATION = 5 * 3 = 15
```

### 冰冻减速效果

```
frozen_speed = original_speed * (1 - FREEZE_REDUCTION) = original_speed * 0.5
```

## Edge Cases

| Scenario | Expected Behavior | Rationale |
|----------|------------------|-----------|
| 玩家点击已死亡怪物 | 忽略点击，守卫者不移动 | 防止攻击无效目标 |
| 守卫者移动途中目标死亡 | 守卫者返回原位，攻击取消 | 不浪费攻击时间 |
| 冰火法杖分裂弹击中同一目标 | 两枚弹均造成伤害，两个状态效果叠加 | 双弹双效，奖励精准瞄准 |
| 灭世之剑全屏扇形中有玩家树木 | 不对树木造成伤害（友军免疫） | 防止自伤 |
| 购买武器时金币不足 | 商店按钮变灰，无法购买，显示"金币不足" | 清晰反馈 |
| 冻结的怪物被燃烧 | 冻结立即解除，燃烧效果应用 | 冰火不兼容，后者覆盖前者 |

## Tuning Knobs

| Parameter | Current Value | Safe Range | Effect of Increase | Effect of Decrease |
|-----------|--------------|------------|-------------------|-------------------|
| BASE_ATTACK_COOLDOWN | 0.5s | 0.2-1.0s | 攻击节奏更慢，压力更大 | 攻击太快显得混乱 |
| 普通剑伤害 | 25 | 15-40 | 初期更强，经济压力减小 | 初期更弱，更需要早买武器 |
| 火焰大剑价格 | 150 | 100-250 | 后期才能买到 | 可以早期入手 |
| 冰火法杖价格 | 300 | 200-500 | 只有效率玩家才能买 | 几乎人人都能买 |
| GUARDIAN_SPEED | 200 px/s | 120-300 px/s | 手感更快，操作更爽 | 手感更重，有策略感 |
| 连击奖励阈值 | 3次 | 2-5次 | 连击奖励更难触发 | 连击奖励很容易触发 |

## Game Feel

### Feel Reference
目标手感：《植物大战僵尸》中豌豆射手的爽快单体攻击感，同时加入《Hades》中武器特效的视觉冲击力。每次攻击要让玩家感受到"我打到了"，不能有延迟感或虚浮感。

灭世之剑参考：《塞尔达传说》的仙女剑气横扫，要有"一击清场"的史诗感。

### Input Responsiveness

| Action | Max Input-to-Response Latency (ms) | Frame Budget (at 60fps) | Notes |
|--------|-----------------------------------|------------------------|-------|
| 鼠标点击怪物 | 50ms | 3帧 | 守卫者立即开始移动 |
| 攻击动画触发 | 100ms | 6帧 | 到达目标后立即攻击 |
| 冷却结束反馈 | 即时 | — | 冷却结束时立即可再次点击 |

### Impact Moments

| Impact Type | Duration (ms) | Effect Description | Configurable? |
|-------------|--------------|-------------------|---------------|
| 普通剑击中顿帧 | 50ms | 轻微画面停顿 | Yes |
| 火焰大剑击中顿帧 | 80ms | 更强顿帧+屏幕轻微震动 | Yes |
| 灭世之剑全屏斩 | 400ms | 全屏闪白→剑气横扫→震动 | Yes |
| Boss击杀 | 800ms | 慢动作+爆炸+满屏粒子 | Yes |

## Acceptance Criteria

- [ ] 左键点击怪物，守卫者飞向目标并攻击
- [ ] 攻击后有0.5秒冷却，冷却期间无法攻击
- [ ] 购买武器消耗对应金币，金币不足时无法购买
- [ ] 每种武器的特效与规格表一致
- [ ] 燃烧/冻结状态按照设计持续时间正确运作
- [ ] 第5波开始时自动赠予灭世之剑
- [ ] 灭世之剑攻击范围覆盖扇形150°内所有怪物
- [ ] Performance: 攻击伤害计算在1帧内完成
