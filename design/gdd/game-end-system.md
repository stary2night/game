# 胜利/失败系统 (Game End System)

> **Status**: Draft
> **Author**: Game Designer
> **Last Updated**: 2026-05-07
> **Implements Pillar**: 环保寓意清晰

## Summary

游戏有两种结局：核心HP归零→失败（显示环保提示语）；5波全清→胜利（过场动画：玩家率众工人拆除工厂，大地重返绿色）。结局动画是游戏环保寓意的最直接表达。

> **Quick reference** — Layer: `Feature` · Priority: `Alpha` · Key deps: `wave-system.md, ecology-system.md`

---

## Overview

结局系统监听两个触发条件：核心HP归零（失败）和第5波清除（胜利）。失败画面简短，鼓励重玩；胜利动画完整，是全游戏的高潮，通过拆厂过场强化环保主题，让玩家感受到实质性的改变。

---

## Player Fantasy

**胜利**：我领导工人拆掉了污染工厂，看着黑烟消散、大地重新变绿，感受到了真正改变世界的力量。

**失败**：失败不是惩罚，而是一次提醒——环保需要不断努力，不能放弃。

---

## Detailed Design

### Core Rules

**失败触发**
1. 纯净核心HP归零时立即触发失败
2. 所有游戏逻辑暂停（怪物停止移动，精灵停止种树）
3. 显示失败画面：核心崩溃动画（2s）→ 灰色覆盖全屏（1s）→ 失败面板

**失败面板内容**
- 标题："领地沦陷"
- 随机环保提示语（见提示语列表）
- 当前波次和击杀数统计
- 两个按钮：[再次挑战] [返回主菜单]

**胜利触发**
1. 第5波所有怪物消灭后触发胜利
2. 游戏逻辑继续运行（精灵继续种树，作为过场背景）
3. 触发胜利过场动画序列

**胜利过场动画序列**（总时长约15秒）
1. 全屏提示"工厂已被削弱，号召工人！"（2s）
2. 过场：玩家守卫者走向右侧工厂，身后跟随一排工人（卡通小人，手持工具）（4s）
3. 工厂开始震动，烟囱倒塌，工厂外墙碎裂（拆除动画）（4s）
4. 黑烟消散，背景从灰黄色渐变为蓝天白云（3s）
5. 领地全景：树木茂盛，精灵欢快飞舞，核心发出明亮光芒（2s）
6. 结局文字淡入："地球感谢你的守护 🌍" + 环保行动号召语

**结局面板内容**
- 游戏标题"绿盾卫士"
- 本局统计：击杀数、金币收入、树木种植数、波次清除数
- 两个按钮：[再次挑战] [分享成绩]

### 环保提示语列表（随机展示）
- "全球每年有150亿棵树被砍伐，种下一棵树从现在开始。"
- "工业排放是全球变暖的主因之一，选择清洁能源，我们每个人都可以行动。"
- "地球上的树木在过去100年中减少了46%，植树护树需要所有人的参与。"
- "一棵成年树每年可吸收约22公斤CO₂，你今天种树了吗？"
- "气候变化不是遥远的威胁，它正在影响每一个地区、每一个生命。"

## Edge Cases

| Scenario | Expected Behavior | Rationale |
|----------|------------------|-----------|
| 第5波最后一只怪物死亡的同时核心HP归零 | 触发胜利（怪物消灭优先） | 最后一击同时发生时，玩家应获得胜利 |
| 胜利过场动画期间玩家点击 | 跳过过场，直接显示结局面板 | 尊重玩家时间 |
| 失败后立即重玩 | 完全重置所有状态：金币、波次、HP、树木 | 确保每局公平 |

## Visual/Audio Requirements

| Event | Visual Feedback | Audio Feedback | Priority |
|-------|----------------|---------------|----------|
| 失败触发 | 核心爆炸，全屏红色闪光，灰色渐变覆盖 | 悲伤短音效，BGM停止 | HIGH |
| 胜利过场-工人出征 | 可爱工人小人行进 | 进行曲 | HIGH |
| 胜利过场-工厂倒塌 | 工厂砖块飞散，烟囱慢动作倒塌 | 轰隆倒塌声+欢呼声 | CRITICAL |
| 胜利过场-大地变绿 | 颜色从灰黄渐变到清新蓝绿 | 欢乐胜利BGM | CRITICAL |
| 结局画面 | 星光粒子漂浮，主题色渐变背景 | 胜利主题曲 | HIGH |

## Acceptance Criteria

- [ ] 核心HP归零时游戏暂停，显示失败面板
- [ ] 失败面板显示随机环保提示语和本局统计
- [ ] 第5波清除后触发胜利过场动画序列
- [ ] 过场动画可跳过
- [ ] 胜利结局面板显示完整统计数据
- [ ] 重玩按钮完全重置游戏状态
