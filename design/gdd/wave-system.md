# 波次入侵系统 (Wave Invasion System)

> **Status**: Draft
> **Author**: Game Designer
> **Last Updated**: 2026-05-07
> **Implements Pillar**: 自然与人的协力

## Summary

共5波怪物从右侧工厂出发，由右向左横穿领地攻击树木和纯净核心。每波怪物数量和种类递增，第4、5波出现Boss级敌人。波与波之间有短暂休息时间，玩家可在此期间升级。

> **Quick reference** — Layer: `Core` · Priority: `MVP` · Key deps: `enemy-ai-system.md, economy-system.md`

---

## Overview

波次系统定义游戏节奏。工厂在右侧持续运作，按照预设时间表生产怪物。怪物排成数路纵队从右向左推进，遇到树木优先攻击树木，树木清除后继续前进攻击核心。消灭所有5波怪物并存活则胜利。

---

## Player Fantasy

玩家应感受到：每一波都是一场有始有终的战役，波间的喘息让人既放松又充满期待。越到后面越紧张，Boss登场时既震撼又有决一死战的刺激感。

---

## Detailed Design

### Core Rules

1. 游戏开始后，工厂右侧出现怪物生成点
2. 波次按顺序触发，当前波所有怪物消灭后，等待 `WAVE_INTERVAL = 8s` 再开始下一波
3. 一共5波，第5波消灭后触发胜利流程
4. 每波开始前屏幕显示波次提示："第X波 / 5"
5. 第4波和第5波各有一个Boss，Boss血量和伤害远超普通怪物
6. 怪物行进路线：从最右侧固定X坐标出发，向左移动，寻找并攻击路径上的树木，之后直奔核心

### 波次配置表

| 波次 | 怪物组成 | 总数 | 特殊规则 |
|------|----------|------|----------|
| Wave 1 | 扭头怪 ×8 | 8 | 新手引导波，速度较慢 |
| Wave 2 | 扭头怪 ×6 + 山羊怪 ×4 | 10 | 引入山羊怪，有冲锋机制 |
| Wave 3 | 扭头怪 ×4 + 山羊怪 ×4 + 骷髅 ×4 | 12 | 引入骷髅远程怪，开始毒气攻击 |
| Wave 4 | 扭头怪 ×6 + 山羊怪 ×4 + 骷髅 ×4 + mini-Boss ×1 | 15 | mini-Boss：污染机甲 |
| Wave 5 | 山羊怪 ×6 + 骷髅 ×6 + mini-Boss ×2 + 大Boss ×1 | 15 | 大Boss：巨型污染魔王；同时赠予灭世之剑 |

### 怪物类型规格

**扭头怪 (Twist-Head Monster)**
- 外观：橙色球形身体，脑袋会360度扭转，卡通丑萌风格
- HP: 60 / 伤害: 8/s / 速度: 60 px/s
- 攻击方式：近战，接触到树木或核心时持续造成伤害
- 特效：被击败时身体旋转炸开，冒出橙色烟雾

**山羊怪 (Goat Monster)**
- 外观：灰色山羊，头上长着齿轮状金属角，眼睛是工业仪表盘
- HP: 100 / 伤害: 15（冲锋）/ 速度: 40 px/s（平时），120 px/s（冲锋）
- 攻击方式：每5秒对前方进行一次冲锋，冲锋时造成高额伤害和击退
- 特效：冲锋时身体前倾，留下烟尘轨迹；被击败时齿轮角飞散

**骷髅 (Skeleton)**
- 外观：穿着破旧工人服的骷髅，手持毒气喷射器
- HP: 50 / 伤害: 5/s（毒气持续伤害）/ 速度: 35 px/s
- 攻击方式：远程喷射毒气，射程150px，对前方扇形区域的树木和核心造成伤害，同时增加该区域毒气浓度
- 特效：绿色毒气喷射动画；被击败时工人服飘落，骷髅碎裂

**mini-Boss — 污染机甲 (Pollution Mech)**
- 外观：4米高的工业机甲，烟囱冒黑烟，身上有污染液体流淌
- HP: 400 / 伤害: 20/s（近战）+ 范围炸弹每10秒一次（伤害50，半径80px）
- 速度: 25 px/s
- 特效：行走时地面震动；炸弹爆炸时烟雾+火焰粒子

**大Boss — 巨型污染魔王 (Giant Pollution Overlord)**
- 外观：巨型污染怪兽，由工业垃圾和污染物构成，有3只眼睛，发出毒绿色光芒
- HP: 1500 / 3个阶段
  - 阶段1 (HP 100%-60%)：近战冲击，每次伤害 30
  - 阶段2 (HP 60%-30%)：释放小型污染怪（2只扭头怪），同时近战
  - 阶段3 (HP 30%-0%)：全场毒气覆盖（所有树木HP持续-3/s），狂暴攻击伤害×1.5
- 入场时：全屏震动特效，背景音乐切换为Boss战主题

### States and Transitions

**波次状态机**

| State | Entry Condition | Exit Condition | Behavior |
|-------|----------------|----------------|----------|
| 等待 (Waiting) | 游戏开始 / 上一波结束 | 计时器归零 | 显示倒计时，玩家可升级 |
| 活跃 (Active) | 计时器归零 | 当前波所有怪物死亡 | 持续生成怪物，怪物行进 |
| 胜利 (Victory) | 第5波所有怪物死亡 | — | 触发胜利过场动画 |

### Interactions with Other Systems

| System | Interaction |
|--------|-------------|
| 怪物AI系统 | 波次系统实例化怪物，AI系统控制怪物移动和攻击 |
| 金币经济系统 | 怪物死亡时向经济系统发送金币奖励事件 |
| 战斗武器系统 | 玩家武器对怪物造成伤害，减少波次中怪物数量 |
| 生态循环系统 | 怪物攻击树木和核心，干扰生态系统 |
| 毒气扩散系统 | 骷髅攻击增加毒气浓度 |

## Formulas

### 金币奖励公式

```
coin_reward = BASE_COIN * WAVE_MULTIPLIER[wave_number] * ENEMY_TYPE_MULTIPLIER
```

| Variable | Type | Range | Source | Description |
|----------|------|-------|--------|-------------|
| BASE_COIN | int | 10 | 配置 | 基础金币奖励 |
| WAVE_MULTIPLIER | float[] | [1.0, 1.1, 1.2, 1.5, 2.0] | 配置 | 各波次金币倍率 |
| ENEMY_TYPE_MULTIPLIER | float | 1.0-10.0 | 配置 | 怪物类型倍率（Boss=10） |

**金币奖励参考值**:
- 扭头怪：10金币
- 山羊怪：18金币
- 骷髅：15金币
- mini-Boss：120金币
- 大Boss：300金币

## Edge Cases

| Scenario | Expected Behavior | Rationale |
|----------|------------------|-----------|
| 怪物数量过多导致拥堵 | 怪物之间保持最小间距50px，相互推开 | 防止怪物堆叠 |
| 大Boss死亡同时玩家HP归零 | 判定玩家胜利（Boss先死） | Boss击败为优先条件 |
| 波间时间内玩家打开商店 | 商店正常响应，波间计时继续 | 不暂停计时 |
| 第5波开始时玩家无法接收灭世之剑（背包满） | 自动替换当前武器 | 保证Boss战机制 |

## Tuning Knobs

| Parameter | Current Value | Safe Range | Effect of Increase | Effect of Decrease |
|-----------|--------------|------------|-------------------|-------------------|
| WAVE_INTERVAL | 8s | 5-15s | 玩家有更多准备时间 | 节奏更紧张 |
| Wave 1怪物数 | 8 | 5-12 | 开局压力更大 | 新手更友好 |
| mini-Boss HP | 400 | 200-600 | Boss战时间更长 | Boss战太短没感觉 |
| 大Boss HP | 1500 | 1000-2500 | 终局高潮更持久 | 终局结束太快 |
| 大Boss阶段3毒气伤害 | 3/s | 1-6/s | 树木快速枯死，紧张感强 | 树木保护时间更长 |

## Acceptance Criteria

- [ ] 5波按顺序触发，每波之间有8秒间隔
- [ ] 每波怪物组成与配置表一致
- [ ] 第5波开始时玩家自动获得灭世之剑
- [ ] 大Boss有3个阶段，每个阶段行为不同
- [ ] 大Boss死亡后触发胜利流程
- [ ] 金币奖励与配置表一致
