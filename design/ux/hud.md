## 顶部 HUD 元素

### 核心HP条
- 位置：左上角
- 样式：绿色血条（宽120px，高16px），HP数值文字在条上
- 颜色变化：> 50% 绿色 / 25%-50% 黄色 / < 25% 红色+闪烁
- 附带：核心小图标（绿色宝石样式）

### 精灵计数
- 位置：核心HP条右侧
- 样式：精灵图标 + "当前/最大" 数字（如"3/8"）

### 树木计数
- 位置：精灵计数右侧
- 样式：树木图标 + 存活数字

### 波次指示器
- 位置：顶部右侧
- 样式：5个圆点，已完成的点为实心绿色，当前波为橙色闪烁，未到的为灰色
- 文字："第 X / 5 波"

### 金币计数
- 位置：顶部最右侧
- 样式：金币图标 + 数字，获得金币时数字跳动动画

## 底部 HUD 元素

### 武器槽
- 位置：左下角
- 样式：武器图标（60×60px），当前武器名称文字
- 冷却圈：图标上方顺时针覆盖的半透明暗色圆形，攻击后从满到空

### 商店按钮
- 位置：底部中央
- 样式：购物车图标 + "商店"文字，绿色圆角按钮
- 状态：可购买新物品时按钮金色发光边框

### 波次倒计时
- 位置：底部右侧
- 样式：沙漏图标 + 秒数（仅波间休息期间显示）
- 文字："下一波：Xs"

## 商店面板（覆盖层）

```
┌──────────────── 商店 ────────────────┐
│ 武器                                  │
│ [🔥火焰大剑]  150金  [购买]           │
│ [❄寒冰大剑]  200金  [购买]           │
│ [🌀冰火法杖]  300金  [购买]          │
│                                       │
│ 树木升级                              │
│ [🌳强化橡树]  100金  [购买]          │
│ [✨神圣生命树] 200金  [购买]         │
│                                       │
│              [关闭]                   │
└───────────────────────────────────────┘
```

- 商店为半透明背景，不完全遮挡游戏画面（透明度60%）
- 金币不足的选项：灰色按钮+红色文字"金币不足"
- 已购买/已满级：按钮显示"✓已拥有"或"已满级"，不可再点击
- 商店打开时游戏不暂停

## 视觉设计原则

1. **卡通风格**：圆角、粗描边、饱和度高的颜色
2. **信息密度适中**：核心信息（HP、波次、金币）始终可见
3. **不遮挡战场**：HUD集中在顶部和底部，中间战场区域完全留出
4. **状态变化清晰**：重要变化（HP低、可买东西）用颜色和动画即时反馈

## Acceptance Criteria

- [ ] 所有顶部HUD元素实时更新
- [ ] 商店面板在游戏不暂停的情况下可操作
- [ ] 冷却圈准确反映攻击冷却进度
- [ ] HP条颜色在三个阈值点正确切换
- [ ] 波次倒计时仅在波间显示
