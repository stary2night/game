# Art Direction: 绿盾卫士

*Created: 2026-05-08*  
*Status: Working Draft*

---

## 1. Visual Core

### World Tone
- **方向**：卡通战斗
- **气质关键词**：轻松、活泼、清晰、环保、对抗但不压抑
- **避免方向**：写实废土、暗黑末日、沉重工业恐怖、过度复杂纹理

### Camera / Map Perspective
- **方向**：轻倾斜俯视
- **参考感受**：类似 `Brawl Stars` 的可读性，但保留更柔和的自然环境气质
- **目标效果**：
  - 地图有明确纵深感
  - 玩家、树木、怪物、障碍之间层级一眼能分清
  - 画面既像战场，也像一个可爱的生态小世界

### Character Proportion
- **方向**：Q版
- **比例建议**：
  - 头部偏大
  - 身体简化
  - 武器轮廓夸张
  - 怪物剪影一眼可辨认

## 2. Art Style Rules

### Shape Language
- **我方阵营**：圆润、柔和、叶片感、流线感
- **污染阵营**：尖角、锯齿、烟雾感、破碎感
- **中立地形**：块面清晰，少细碎噪点

### Rendering Style
- **主风格**：卡通手绘感 + 干净色块
- **明暗关系**：中高明度，少用纯黑
- **边缘处理**：适度描边，不要太硬太粗
- **细节密度**：中等偏低，优先保证战斗可读性

### FX Style
- **攻击特效**：明亮、夸张、短促
- **污染特效**：紫灰、墨绿、雾状扩散
- **净化特效**：浅绿、金黄、青蓝，带发光颗粒

## 3. Color Direction

### Overall Principle
- **目标**：轻松活泼、容易吸引小朋友、同时保留对抗张力
- **策略**：
  - 领地和自然区域用清新明亮色
  - 工厂和污染区域用偏灰紫、灰棕、脏绿
  - 战斗反馈和按钮用高饱和强调色

### Suggested Palette

#### Nature / Friendly
- 草地主色：`#7FD36B`
- 草地高光：`#A8E88C`
- 深草阴影：`#4E9F46`
- 树木主色：`#38B66B`
- 树干色：`#8B5A2B`
- 河流主色：`#5BC7E8`
- 河流高光：`#A9EDFF`

#### Pollution / Factory
- 污染紫灰：`#6F5B7E`
- 烟雾灰：`#5F646E`
- 工厂棕灰：`#7A6858`
- 危险橙：`#F08A24`
- 深污染绿：`#5B7A3A`

#### Combat / UI Accent
- 主按钮绿：`#2FCC71`
- 金币黄：`#FFD54A`
- 命中橙：`#FF8B3D`
- 冰系蓝：`#7ED6FF`
- 火系橙红：`#FF6A3D`
- 警示红：`#F25555`

### Palette Rule
- 同屏主色不要超过 5 个大类
- 污染色永远不要比自然色更鲜亮
- 重要交互物必须比背景亮一档

## 4. Map Redesign Direction

### Current Problems
- 河流过宽，切断感太强
- 地图地表变化太少，像原型底板
- 领地与工厂分区虽然明确，但景观叙事不够
- 地图缺少“儿童会喜欢观察”的小景物层次

### New Map Goal
- 看起来像一个“正在被污染侵蚀的绿色乐园”
- 左侧领地要有生机、童趣和守护感
- 右侧工厂区要有侵入感，但不能太恐怖
- 中间过渡区要体现生态与污染对抗
- 地势上形成“我方高、敌方低”的空间关系，增强守护与压境的感觉

### New Layout Proposal

#### Left: Guardian Territory
- 位于山地高处或高坡平台
- 草地为主
- 带少量花丛、蘑菇、小石块、灌木
- 纯净核心周围有神圣环形地纹
- 树木种植格子用更自然的地表块提示，而不是生硬网格感
- 地表不完全平坦，可有柔和高低起伏、台地边缘、草坡过渡
- 视觉感受应接近“绿树茵茵、花草繁茂的守护圣地”

#### Center: Conflict Belt
- 是自然与污染之间的明确过渡地带
- 河流缩窄为可阅读的弯曲溪流或浅河道
- 加入木桥 / 石桥 / 踏石，增强地图记忆点
- 地面从鲜绿逐渐过渡到灰黄和污染斑块
- 可加入几块被污染侵蚀的草地裂纹
- 地貌从高处缓慢向低处倾斜，形成从山地到平原的地势下降
- 植被密度逐渐降低：鲜花、灌木、绿草减少，裸土、黄草、碎石增加
- 这是地图叙事最重要的区域，要让玩家一眼看到“污染正在推进”

#### Right: Factory Zone
- 位于相对低处的平原工业区
- 保留工厂压迫感，但做成卡通工业区
- 管道、烟囱、废料桶、机械平台要简化成大形块
- 地面使用棕灰、暗紫灰、脏黄混合
- 敌人出生点要像“污染喷口”或“工业裂缝”，而不是凭空出现
- 整体应从草地彻底过渡成荒地、污地、工业废土
- 地面更平整、更压实，和左侧富有生命感的高地区形成强烈对比

### River Adjustment
- 宽度缩小到现在的约 `45% - 60%`
- 形状改为有节奏的弯曲
- 两岸增加草边、石头、浅滩变化
- 河流更像地图装饰和空间引导，而不是粗暴分割线
- 河流可以顺着坡势从高处向低处流动，强化整体地形逻辑

### Height / Terrain Rule
- 地图不能是完全平的
- 左高右低是核心地貌规则
- 高差表现不靠真实3D，而靠以下方式暗示：
  - 地块边缘的明暗层次
  - 台地边缘和坡面
  - 物体脚下投影变化
  - 植被密度和地表材质变化
  - 桥梁、溪流、坡道带来的空间导向

### Ecological Contrast Rule
- 我方区域：繁盛、湿润、明亮、有生命力
- 过渡区域：绿色减弱、土壤裸露、被污染侵蚀
- 敌方区域：荒地化、灰化、工业化、污染化
- 这种差异必须远看就成立，不依赖近看细节

### P0: Must Define First

- 树木升级外观差异
- 攻击特效基础包
- 污染扩散视觉层

### P2: Produce Later
- 开始界面插画
- 胜利 / 失败演出图层
- 装饰性小动物、花草、环境动画

## 6. Implementation Advice

### Current Stage Recommendation
- **先定视觉规则，不急着做完整资产包**
- **先重做地图概念和主要角色造型**
- **再同步调整 UI 和特效**

### Best Next Step
1. 先产出一张新的地图方向稿
2. 再确定守卫者、树木、核心、工厂的统一造型语言
3. 最后再批量替换怪物和 UI

## 7. Immediate Decisions Locked

- 世界观气质：卡通战斗
- 地图视角：轻倾斜俯视
- 角色比例：Q版
- 配色方向：轻松活泼、适合儿童、明亮自然对抗污染
- 地图改造重点：缩窄河流、增加景观层次、强化领地/污染/工厂的视觉叙事
- 地势关系：我方山地高处，敌方平原低处，中间有生态衰败过渡带
