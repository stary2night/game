# 游戏设计支柱 — 绿盾卫士

> **Status**: Approved
> **Last Updated**: 2026-05-07

## 核心支柱

### 支柱 1：自然与人的协力 (Nature-Human Synergy)
自动化生态系统（精灵/树木循环）和玩家主动战斗（武器/攻击）必须同等重要、相互依存，任何一方单独都不足以完成游戏。

*设计检验*：若某设计让玩家可以完全忽略生态系统通关，或完全忽略战斗通关，则违反此支柱。

### 支柱 2：卡通趣味感 (Cartoon Fun First)
所有视觉元素（怪物、特效、角色、UI）必须保持欢快卡通风格，色彩鲜明，造型友善，让全年龄段玩家都能接受和喜爱。

*设计检验*：若任何视觉元素让8岁儿童感到恐惧或不适，需重新设计。

### 支柱 3：环保寓意清晰 (Clear Eco Message)
游戏的每个机制、结局和叙事元素都应能对应到现实世界的环保行动，让玩家在游玩中潜移默化地接受环境保护的理念。

*设计检验*：若有玩家通关后不理解游戏在讲环境保护，需加强叙事提示和视觉隐喻。

## 设计反支柱（明确排除项）

- **不做复杂策略**：不做多路线建造、复杂资源管理，保持操作简单，1分钟内可上手
- **不做黑暗末日风**：不渲染污染的残酷，不让失败显得绝望，保持积极向上的基调
- **不做无尽模式**：游戏有明确的5波结局和拆厂过场，不做无休止的关卡
- **不做多人竞技**：单人体验，专注于个人叙事和环保共鸣

## 设计决策参考

当遇到设计冲突时，用这三个支柱来做决策：

| 场景 | 决策 |
|------|------|
| 树木是否应该主动攻击怪物？ | ❌ 违反支柱1：会削弱玩家武器的重要性 |
| 怪物是否需要血腥死亡特效？ | ❌ 违反支柱2：改为卡通式消散爆炸效果 |
| 失败时是否显示"地球末日"画面？ | ❌ 违反支柱2+3：改为温和的环保提示语 |
| 波间是否可以直接跳过等待？ | ✅ 支持支柱2：减少无聊等待，提升趣味感 |
| 胜利后是否显示实际环保数据？ | ✅ 支持支柱3：增加现实联系，强化寓意 |
